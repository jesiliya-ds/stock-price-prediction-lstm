# Time-Series Stock Price Prediction using LSTM

This project focuses on predicting stock prices using a Long Short-Term Memory (LSTM) deep learning model. The model analyzes historical stock market data and learns patterns in time-series data to forecast future stock price trends.

## Technologies Used
- Python
- TensorFlow
- Pandas
- NumPy
- Matplotlib

## Project Description
In this project, historical stock market data is used to train an LSTM-based deep learning model. Data preprocessing, normalization, and sequence generation techniques are applied before training the model.

The trained model predicts future stock prices based on previous trends in the dataset.

## Model Workflow
1. Data Collection
2. Data Preprocessing
3. Time-Series Data Preparation
4. Model Training using LSTM
5. Stock Price Prediction
6. Visualization of Results

## Dataset
The dataset used in this project contains historical stock price data (BHARTIARTL.NS). It includes information such as open price, close price, high, low, and volume.

## Output
The model generates predicted stock prices and visualizes them using graphs to compare actual and predicted values.
